# TalentLean - Official Development Repository

AI-powered LatAm staff augmentation platform. Build and scale your Latin American tech teams with full compliance, transparent pricing, and expert vetting.

## Features

- AI-driven candidate vetting and matching
- Full EOR (Employer of Record) services
- Transparent flat-fee pricing model
- LatAm talent network
- Integrated payroll and compliance management

## Tech Stack

- React 18 + Vite
- TailwindCSS
- React Router v7
- Base44 SDK for backend integration
- Radix UI components

## Development

```bash
npm install
npm run dev
```

## Production

Deployed via Vercel from upstream repository.
